## pvm

slightly more convenient venv switching

## install
requires python >= 3.6.5

```
cd ./pvm

python ./profile.py >> ~/.profile
source ~/.profile

pvm --help
```

### create new venv dir
```
pvm init <dir>
```

### us existing venv dir
```
pvm use <dir>
```
